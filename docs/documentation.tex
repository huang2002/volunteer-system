\section{使用系统}

\subsection{启动方法}

\begin{itemize}
    \item[Windows系统] 双击\textit{启动.bat}。
    \item[MacOS/Linux系统] 运行\textit{启动.bash}。
\end{itemize}

\subsection{使用系统}

运行启动脚本后，应该会打开一个命令行窗口，显示后台程序的信息，
并自动打开浏览器窗口，展示管理系统界面。

需要退出时，直接关闭浏览器和命令行窗口即可。

\subsection{数据保存}

数据文件以CSV格式存放在\textit{data}文件夹中，
文件名称应为\textit{年级.csv}，
其中\textit{年级}是两位阿拉伯数字。
